# /สร้างวิดีโอ — Generate TikTok Video from tiktok.md

สร้างไฟล์วิดีโอ `.mp4` สำหรับ TikTok จากรูปสินค้าและ script ใน `tiktok.md`
ถ้า FFmpeg ไม่พร้อม → สร้างคู่มือตัดต่อ CapCut แทน

Arguments: $ARGUMENTS
- ว่างเปล่า → สร้างทุกสินค้าที่มี tiktok.md แต่ยังไม่มี video.mp4 / capcut.md
- `YYYY-MM-DD` → สร้างเฉพาะสินค้าที่มี post_date ตรงกับวันนั้น
- `{item_id}` → สร้างเฉพาะสินค้านั้น
- `--force` → สร้างใหม่แม้มีไฟล์อยู่แล้ว
- `--sec N` → กำหนดเวลาต่อ scene (default: 2 วินาที)

ตัวอย่าง:
```
/สร้างวิดีโอ 3991346022
/สร้างวิดีโอ 2026-05-17
/สร้างวิดีโอ 3991346022 --force
/สร้างวิดีโอ 3991346022 --sec 3
/สร้างวิดีโอ 3991346022 --force --sec 2
```

---

## ขั้นตอนที่ 1 — ตรวจสอบ FFmpeg

```
node -e "
const {execSync}=require('child_process');
try { execSync('ffmpeg -version',{stdio:'ignore'}); console.log('OK'); }
catch { console.log('NOT_FOUND'); }
"
```

**ถ้าผล `OK`** → ดำเนินการสร้างวิดีโอจริง (ขั้นตอนที่ 2A)
**ถ้าผล `NOT_FOUND`** → แจ้งวิธีติดตั้งแล้วสร้าง CapCut guide แทน (ขั้นตอนที่ 2B)

## ขั้นตอนที่ 2A — FFmpeg พร้อม: สร้างวิดีโอจริง

### หาสินค้าที่ต้องสร้าง

```
node -e "
const fs=require('fs'), path=require('path');
const filter='$ARGUMENTS'.replace('--force','').replace(/--sec\s+\d+/,'').trim();
const force='$ARGUMENTS'.includes('--force');
const dirs=fs.readdirSync('products').filter(d=>fs.existsSync('products/'+d+'/data.json'));
const pending=[];
dirs.forEach(id=>{
  const d=JSON.parse(fs.readFileSync('products/'+id+'/data.json','utf8'));
  if(d.status==='placeholder') return;
  const isDate=/^\d{4}-\d{2}-\d{2}$/.test(filter);
  const isId=/^\d{8,}$/.test(filter);
  if(filter&&isDate&&d.post_date!==filter) return;
  if(filter&&isId&&id!==filter) return;
  if(!fs.existsSync('products/'+id+'/content/tiktok.md')) return;
  const hasVideo=fs.existsSync('products/'+id+'/video.mp4');
  if(hasVideo&&!force) return;
  const imgs=fs.existsSync('products/'+id+'/images')
    ? fs.readdirSync('products/'+id+'/images').filter(f=>f.endsWith('.jpg')).length : 0;
  pending.push({id, postDate:d.post_date||'', title:(d.title||'').substring(0,35), imgs, hasVideo});
});
pending.sort((a,b)=>a.postDate.localeCompare(b.postDate));
pending.forEach(p=>console.log(p.postDate+' | '+p.id+' | '+p.title+' | รูป:'+p.imgs+(p.hasVideo?' [overwrite]':'')));
if(!pending.length) console.log('NONE');
"
```

- ถ้า `NONE` → แจ้ง "วิดีโอครบทุกสินค้าแล้ว"
- ถ้ามีรายการ → แสดงให้ผู้ใช้เห็นแล้วถาม **"ยืนยันสร้างวิดีโอ N รายการไหม?"** รอคำตอบก่อน

### รัน make-tiktok-video.js

```
node make-tiktok-video.js $ARGUMENTS
```

timeout 10 นาทีต่อสินค้า 1 ชิ้น (ขึ้นอยู่กับความยาววิดีโอและความเร็ว CPU)

**Error handling:**
- `ไม่พบ FFmpeg` → ไปขั้นตอน 2B
- `ไม่พบรูปใน images/` → แจ้งให้รัน `/ดึงสินค้า` ก่อน
- `parse tiktok.md ไม่พบ scene` → แจ้งให้รัน `/สร้าง-content` ก่อน
- `FFmpeg scene error` → แสดง error message ให้ผู้ใช้เห็น

## ขั้นตอนที่ 2B — FFmpeg ไม่พร้อม: สร้าง CapCut Guide

แจ้งผู้ใช้:
```
⚠️ ไม่พบ FFmpeg — สร้าง CapCut guide แทน

ติดตั้ง FFmpeg (ฟรี) เพื่อสร้างวิดีโอจริง:
1. ไปที่ https://ffmpeg.org/download.html → Windows → gyan.dev
2. ดาวน์โหลด ffmpeg-release-essentials.zip (~90 MB)
3. แตกไฟล์ → copy ไปที่ C:\ffmpeg\
4. เพิ่ม C:\ffmpeg\bin ใน System PATH:
   Win+R → sysdm.cpl → Advanced → Environment Variables
   → Path → Edit → New → C:\ffmpeg\bin → OK
5. เปิด terminal ใหม่ → ffmpeg -version → ถ้าเห็นเวอร์ชัน = พร้อม
6. รัน /สร้างวิดีโอ อีกครั้ง
```

จากนั้นสร้าง `capcut.md` แทนโดยทำตามขั้นตอนนี้:

### สร้าง capcut.md

อ่าน `products/{item_id}/data.json` + `products/{item_id}/content/tiktok.md` + list รูปใน `products/{item_id}/images/`

สร้างไฟล์ `products/{item_id}/content/capcut.md` ที่มีโครงสร้าง:

```markdown

| ความละเอียด | 1080 × 1920 |

## 🖼️ รายการรูปที่ใช้

{image_table}
```

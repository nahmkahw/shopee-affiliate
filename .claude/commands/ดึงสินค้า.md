# /ดึงสินค้า — Shopee Affiliate Product Scraper

ดึงข้อมูลสินค้าจาก Shopee ทุกรายการใน `input/urls.txt`
ข้ามสินค้าที่มี data.json แล้วอัตโนมัติ
ทำตามขั้นตอนด้านล่างตามลำดับ อย่าข้ามขั้นตอน

Arguments (ถ้ามี): $ARGUMENTS

---

## ขั้นตอนที่ 1 — ตรวจสอบ Prerequisites

รันพร้อมกัน:
```
node --version && ls input/urls.txt && curl -s http://localhost:9222/json/version 2>&1 | head -c 100
```

- `node` ไม่พบ → แจ้ง "ต้องติดตั้ง Node.js ก่อน ดู GUIDE.md" แล้วหยุด
- `input/urls.txt` ไม่พบ → แจ้ง "ต้องสร้าง input/urls.txt ก่อน ดูตัวอย่างใน GUIDE.md" แล้วหยุด
- Chrome port 9222 ไม่ตอบสนอง → **หยุด** แล้วแสดง:

  ```
  ❌ Chrome ยังไม่ได้เปิดในโหมด debug

  วิธีเปิด:
  1. ปิด Chrome ทั้งหมดก่อน
  2. กด Win+R แล้วพิมพ์:
     "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\ChromeDebug"
  3. Login Shopee ใน Chrome ที่เปิดขึ้นมา
  4. รัน /ดึงสินค้า อีกครั้ง
  ```

## ขั้นตอนที่ 2 — Dry-run แสดงรายการสินค้า

รัน `node scrape.js --dry-run` แล้วแสดงผลให้ผู้ใช้เห็น

ตัวอย่าง output ที่จะเห็น:
```
📋 พบสินค้า 4 รายการ (ดึงใหม่ 2, ข้าม 2):

  1. [2026-05-20] 3991346022  https://s.shopee.co.th/xxx  ⏭ ข้าม
  2. [2026-05-21] 19283435771 https://s.shopee.co.th/xxx
  3. [2026-05-27] 24240277068 https://s.shopee.co.th/xxx
  4. [2026-05-28] 28813756155 https://s.shopee.co.th/xxx  ⏭ ข้าม
```

ถามว่า "ยืนยันดึงข้อมูล N รายการนี้ไหม?" แล้วรอคำตอบ
ถ้าผู้ใช้ต้องการดึงใหม่ทั้งหมด (รวมที่ข้ามด้วย) → ใช้ `node scrape.js --force`

## ขั้นตอนที่ 3 — รัน Scraper

รัน `node scrape.js` (หรือ `--force` ถ้าผู้ใช้ขอ)

timeout 5 นาทีต่อสินค้า 1 ชิ้น

- error "connect" → Chrome ปิดไปแล้ว แจ้งให้เปิดใหม่
- title = "Loading Issue" หรือ "Verify to Continue" → Shopee บล็อก bot ให้แจ้ง:
  ```
  ⚠️ Shopee แสดงหน้า bot-check สำหรับ item_id นั้น
  กรุณาเปิด Chrome แล้วไปที่ URL สินค้าด้วยตัวเอง แก้ CAPTCHA ให้ผ่าน
  แล้วแจ้ง "พร้อม" เพื่อรัน --force ใหม่
  ```

## ขั้นตอนที่ 4 — ตรวจสอบผลลัพธ์

หลัง scraper เสร็จ รัน:
```
node -e "
const fs = require('fs');
const dirs = fs.readdirSync('products').filter(d => fs.existsSync('products/'+d+'/data.json'));
dirs.forEach(id => {
  const d = JSON.parse(fs.readFileSync('products/'+id+'/data.json','utf8'));
  const imgs = fs.readdirSync('products/'+id+'/images').length;
  const warn = [d.price,d.rating,d.shop_name].filter(v=>!v).length;
  console.log('['+d.post_date+'] '+id+': '+((d.title||'').substring(0,35))+' | '+d.price+' | ⭐'+d.rating+' | รูป:'+imgs+(warn?' ⚠️':''));
});
"
```

แสดงตาราง summary พร้อมแจ้ง field ที่เป็น null (ถ้ามี)

# /ดึงสินค้า-อัตโนมัติ — ดึงรายการสินค้าจาก Shopee Affiliate Portal

เข้า https://affiliate.shopee.co.th/offer/product_offer ผ่าน Chrome
คลิกปุ่ม "เอาลิงค์" ของแต่ละสินค้า → บันทึก affiliate link + post_date ลง `input/urls.txt`
ข้ามสินค้าที่มี item_id อยู่ใน urls.txt แล้วอัตโนมัติ

Arguments: $ARGUMENTS
- ว่างเปล่า → ดึงทั้งหน้า ข้ามที่มีแล้ว
- `--dry-run` → แสดงรายการที่จะเพิ่ม ไม่บันทึก

ตัวอย่าง:
```
/ดึงสินค้า-อัตโนมัติ
/ดึงสินค้า-อัตโนมัติ --dry-run
```

## ขั้นตอนที่ 1 — ตรวจสอบ Prerequisites

รันพร้อมกัน:
```
node --version && curl -s http://localhost:9222/json/version 2>&1 | head -c 100
```

- `node` ไม่พบ → แจ้ง "ต้องติดตั้ง Node.js ก่อน" แล้วหยุด
- Chrome port 9222 ไม่ตอบสนอง → **หยุด** แล้วแสดง:

  ```
  ❌ Chrome ยังไม่ได้เปิดในโหมด debug

  วิธีเปิด:
  1. ปิด Chrome ทั้งหมดก่อน
  2. กด Win+R แล้วพิมพ์:
     "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\ChromeDebug"
  3. Login Shopee Affiliate ที่ https://affiliate.shopee.co.th
  4. รัน /ดึงสินค้า-อัตโนมัติ อีกครั้ง
  ```

> ⚠️ **สำคัญ:** ต้องเปิดหน้า `https://affiliate.shopee.co.th/offer/product_offer` ด้วยตัวเองใน Chrome
> **ห้าม** ให้ script navigate ให้ เพราะ Shopee จะตรวจจับเป็น bot → CAPTCHA
> รอให้สินค้าโหลดครบก่อน แล้วค่อยรัน script

## ขั้นตอนที่ 3 — รัน Scraper

```
node scrape-offers.js $ARGUMENTS
```

timeout 3 นาที (ทั้งหมด)

**Error handling:**
- `ยังไม่ได้ login` → แจ้งให้ไป login ที่ https://affiliate.shopee.co.th แล้วรันใหม่
- `ไม่พบปุ่ม "เอาลิงค์"` → ดู Troubleshooting

## ขั้นตอนที่ 4 — ตรวจสอบผลลัพธ์

แสดง urls.txt ที่อัปเดตแล้ว:
```
node -e "
const fs = require('fs');
const lines = fs.readFileSync('input/urls.txt','utf8').split('\n').filter(l=>l.trim()&&!l.startsWith('#'));
console.log('รายการทั้งหมดใน urls.txt: ' + lines.length + ' รายการ\n');
lines.forEach((l,i)=>{
  const p = l.split('|').map(s=>s.trim());
  const m = p[0] && p[0].match(/\/(\d+)\/(\d+)/);
  if(m) console.log((i+1)+'. ['+p[2]+'] '+m[2]+' | '+p[1]);
});
"
```

## 🔧 Troubleshooting

### ไม่พบปุ่ม "เอาลิงค์"
ชื่อปุ่มบน Shopee Affiliate อาจเปลี่ยนแปลง ให้:
1. เปิด Chrome ไปที่ https://affiliate.shopee.co.th/offer/product_offer
2. ดูชื่อปุ่มที่ใช้ดึง affiliate link (อาจเป็น "Get Link", "รับลิงก์" ฯลฯ)
3. แจ้งชื่อปุ่มนั้น → จะอัปเดต `BTN_KEYWORDS` ใน scrape-offers.js ให้

### อ่านลิงค์ไม่ได้ (modal มีแต่ว่าง)
- modal อาจใช้ iframe → scrape-offers.js ยังไม่รองรับ iframe
- แจ้งให้ทราบ → จะเพิ่ม iframe support

### ต้องการเพิ่มสินค้าด้วยมือ
เพิ่มบรรทัดนี้ลง `input/urls.txt`:
```
https://shopee.co.th/product/{shop_id}/{item_id}  | https://s.shopee.co.th/xxxxx  | YYYY-MM-DD
```

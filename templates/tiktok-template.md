# TikTok Template — Shopee Affiliate

## Goal
สคริปต์วิดีโอ 30-60 วินาที + caption + hashtag สำหรับโพสต์
วิดีโอสร้างอัตโนมัติด้วย `make-tiktok-video.js` — Edge TTS อ่านเสียงจากคอลัมน์ VOICEOVER โดยตรง

## ⚠️ กฎสำคัญมากสำหรับคอลัมน์ VOICEOVER

**VOICEOVER = ประโยคพูดจริงๆ ที่ TTS จะอ่านออกเสียง**

✅ ถูก — เขียนประโยคที่พูดได้เลย:
> "วันนี้เอาหนังสือติวสอบท้องถิ่น ภาค ก ปี 69 มาให้ดู ราคาลดไปถึง 70% เลย"

❌ ผิด — ห้ามใส่คำอธิบาย/คำสั่ง/หัวข้อ:
> "Key features", "CTA", "Introduce product", "ไม่แนะนำ 'สวัสดีค่ะ'", "Hook"

**กฎ VOICEOVER:**
- เขียนเป็นประโยคภาษาไทยสมบูรณ์ที่พูดได้ทันที
- ห้ามใส่ bracket [], คำสั่ง, หรือ label ใดๆ
- ห้ามเริ่มด้วย "สวัสดีค่ะ/ครับ"
- ความยาวต้องสัมพันธ์กับช่วงเวลา (0:03 = ~15 คำ, 0:15 = ~50 คำ)
- ใช้ภาษาพูดเป็นกันเอง ไม่เป็นทางการ

## Variables
เหมือน FB/IG + `{{video_length}}` (30/45/60s) + `{{hook_seconds}}` (3 หรือ 5)

## Structure — แบ่ง 4 ส่วน

### Part 1 — Hook (0-3 วินาที) **สำคัญที่สุด**
ถ้า hook ไม่โดน คนเลื่อนผ่านทันที **อย่าเริ่มด้วย "สวัสดีค่ะ/ครับ"**

Hook formula:
- **Pattern interrupt:** "หยุดก่อน ถ้ายังใช้หูฟังแบบเดิมๆ..."
- **Bold claim:** "หูฟัง 590 บาท ที่ดีกว่าตัวพัน"
- **Question:** "หูฟังราคาไม่ถึงพันที่ดีจริงๆ มีไหม?"
- **Show product close-up + price overlay**
- **Before/After:** เสียงรบกวน → เปิด ANC

### Part 2 — Build-up (3-10 วินาที)
ขยาย hook + บอกว่าเรากำลังจะรีวิวอะไร
- "วันนี้เอามาให้ดู XYZ Pro ที่กำลังลดราคาอยู่"
- โชว์ผลิตภัณฑ์ในมือ unbox

### Part 3 — Key Points (10-45 วินาที) **เนื้อหาหลัก**
3-4 จุดเด่น แต่ละจุด 5-10 วินาที + visual demo
- จุดที่ 1 + visual (ใส่ฟังเพลง / เดินผ่านเสียงดัง)
- จุดที่ 2 + visual (โชว์ %แบต)
- จุดที่ 3 + visual (เอาน้ำราด / ใส่ออกกำลังกาย)
- จุดที่ 4 + visual (โชว์ multipoint)

### Part 4 — CTA (45-60 วินาที)
- "ราคาตอนนี้ {{price}} บาท จากปกติ {{original_price}}"
- "ลิงก์ใน bio / โปรไฟล์"
- ปิดด้วย hook ให้กด follow: "กดติดตามจะรีวิวของดี Shopee ตัวอื่นๆ ต่อ"

## Caption Strategy (แสดงใต้วิดีโอ)
สั้นกว่า IG — 50-80 คำพอ

```
[1 ประโยค hook ตอบสนอง video]

[2-3 จุดเด่นย่อ]

ราคา {{price}} (ปกติ {{original_price}})
ลิงก์ใน bio 🔗

#Shopeeaffiliate #[4-8 tags]
```

## Hashtag TikTok (5-10 ตัว แตกต่างจาก IG)

**Algorithm tags (3 ตัว):**
`#fyp` `#fypシ` `#foryou`

*(บางคนบอก fyp ไม่ช่วย แต่ใส่ไว้ปลอดภัยไม่เสียอะไร)*

**Required:**
`#Shopeeaffiliate` `#shopeefinds`

**Topic tags ภาษาไทย (3-5 ตัว):**
`#ของน่าซื้อshopee` `#รีวิวสินค้า` `#บอกต่อ` `#ของมันต้องมี`

**Niche tags:**
ตัวอย่างหูฟัง: `#หูฟังบลูทูธ` `#หูฟังราคาประหยัด`

⚠️ TikTok ไม่ชอบ hashtag spam — 5-8 ตัวพอ น้อยกว่า IG ครึ่ง

## On-Screen Text
TikTok มี caption ในวิดีโอ (overlay text) ที่สำคัญมาก
- ใส่ทุก scene ที่มีเสียงพูด (เผื่อคนดูแบบปิดเสียง = 50%+)
- ตัวใหญ่ อ่านง่าย วาง upper-third (หลีกจุดที่ TikTok วาง UI)
- ใส่ตัวเลขราคา/% ลด ตัวใหญ่ๆ ทุก 3-5 วินาที

## Music
- ใช้ trending sound ของ TikTok (เพิ่ม reach 30-50%)
- เช็ค Discover tab → Sounds ทุกสัปดาห์
- เสียง trend ที่ไม่เกี่ยวกับ topic ก็ได้ ขอแค่ trending

## Tone
- เป็นกันเอง พูดตรงเข้าประเด็น
- พูดเร็ว (TikTok algorithm ชอบเนื้อหาแน่น)
- ห้ามขายตรงเกินไป — TikTok algo ลด reach โพสต์ที่ดู "ขายของ" จัด
- เน้น "review / share / experience" framing มากกว่า "buy now"

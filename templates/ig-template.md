# Instagram Template — Shopee Affiliate

## Goal
แคปชั่น 100-150 คำ + hashtag 15-20 ตัว
**สำคัญ:** IG กดลิงก์ในแคปชั่นไม่ได้ ต้องเตรียม link-in-bio ด้วย (Linktree / Beacons / Shopee อย่างเดียวก็ได้)

## Variables
- `{{product_name}}`, `{{price}}`, `{{original_price}}`, `{{discount_percent}}`
- `{{key_features}}` (3-4 ข้อ)
- `{{rating}}`, `{{review_count}}`
- `{{affiliate_link}}` — ใส่ใน Linktree/bio
- `{{category}}` — สำหรับเลือก hashtag

## Structure (5 ส่วน)

### 1. Hook Line (1 บรรทัด, มี emoji)
สั้น โดน น่าหยุดดู
- "ของถูกใจเดือนนี้ ✨"
- "เจอแล้ว ไอเทมที่หาอยู่ 🛒"
- "บอกต่อก่อนหมดสต็อก 🔥"

### 2. Brief Body (2-3 ประโยค)
อธิบายว่าสินค้าเหมาะกับใคร / ใช้แก้ปัญหาอะไร
เลี่ยงคำว่า "ขาย" "สั่งซื้อ" ตรงๆ ใน body (เก็บไว้ CTA)

### 3. Key Points (3-4 bullet + emoji แต่ละข้อ)
```
🎯 {{feature_1}}
🔋 {{feature_2}}
💧 {{feature_3}}
🎵 {{feature_4}}
```

### 4. Price Tease + CTA
```
💰 {{price}} บาท (ปกติ {{original_price}})
👉 ลิงก์ใน bio หรือคอมเมนต์ "สนใจ" เดี๋ยวส่งลิงก์ให้
```

### 5. Disclosure + Hashtag Block
```
.
.
.
#Shopeeaffiliate
#[15-19 hashtag mix Thai/English/popular/niche]
```

## Tone
- กระชับ punchy ไม่เล่ายาว
- emoji เยอะกว่า FB ได้ แต่ไม่ปะหน้า
- bold word ทำไม่ได้ใน IG ใช้ CAPS น้อยๆ แทนคำเน้น

## Hashtag Strategy (15-20 ตัว แบ่ง 4 ชั้น)

**ชั้น 1 — Required (3 ตัว)**
`#Shopeeaffiliate` `#Shopeeไทย` `#รีวิวสินค้า`

**ชั้น 2 — Mega tag ภาษาไทย (4-5 ตัว, >100k posts)**
`#ของมันต้องมี` `#ของน่าซื้อshopee` `#ไอเทมเด็ด` `#รีวิวของดี` `#บอกต่อ`

**ชั้น 3 — Niche tag เฉพาะสินค้า (5-7 ตัว, 10k-100k posts)**
ตัวอย่างหูฟัง: `#หูฟังบลูทูธ` `#หูฟังไร้สาย` `#TWS` `#earbuds` `#หูฟังANC`

**ชั้น 4 — Long-tail / brand (3-5 ตัว, <10k posts)**
ตัวอย่าง: `#หูฟังราคาประหยัด` `#หูฟังใต้พัน` `#รีวิวหูฟัง2026`

## Carousel Strategy
ถ้าโพสต์เป็น carousel (แนะนำ 5-7 ภาพ):
- ภาพ 1: cover ที่มีข้อความหลัก + ราคา
- ภาพ 2-4: รูปสินค้าจริงจากร้าน (มุมต่างๆ)
- ภาพ 5: feature highlight (text overlay)
- ภาพ 6: รีวิวลูกค้า / rating
- ภาพ 7: CTA "ลิงก์ใน bio"

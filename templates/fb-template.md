# Facebook Template — Shopee Affiliate

## Goal
สร้างโพสต์ Facebook แบบเล่าเรื่อง (storytelling) ที่ engagement ดี
ความยาว 150-300 คำ มีลิงก์ affiliate ท้ายโพสต์

## Variables (Cowork ต้องเติมจาก data.json)
- `{{product_name}}` — ชื่อสินค้า
- `{{price}}` — ราคาปัจจุบัน
- `{{original_price}}` — ราคาก่อนลด
- `{{discount_percent}}` — % ส่วนลด
- `{{key_features}}` — list 3-5 จุดเด่น
- `{{rating}}` — คะแนน (เช่น 4.8)
- `{{review_count}}` — จำนวนรีวิว
- `{{top_review_snippet}}` — รีวิวเด่น (ตัด <60 คำ)
- `{{affiliate_link}}` — affiliate short link
- `{{shop_name}}` — ชื่อร้าน

## Structure (8 ส่วน)

### 1. Hook (1 ประโยค)
ตั้งคำถาม / pain point / สถานการณ์ที่คนเจอจริง
- "ใครเคย..."
- "เบื่อไหม เวลา..."
- "บอกตรงๆ ว่า..."
- "เพิ่งเจอของดีมา ต้องบอกต่อ"

### 2. Story / Context (2-3 ประโยค)
เล่าประสบการณ์/บริบทที่ทำให้สินค้านี้มีค่า ห้ามแต่งรีวิวเอง
ถ้าไม่มีประสบการณ์จริง ใช้ scenario เชิงสมมติว่า "ลองนึกภาพ..."

### 3. แนะนำสินค้า (1-2 ประโยค)
"ขอแนะนำ {{product_name}} จากร้าน {{shop_name}}"

### 4. Key Features (bullet 3-5 ข้อ + emoji)
```
✅ {{feature_1}}
✅ {{feature_2}}
✅ {{feature_3}}
✅ {{feature_4}}
```

### 5. Social Proof (1 ประโยค)
"ตอนนี้รีวิว {{rating}}/5 จาก {{review_count}}+ รีวิว"
หรือยก quote สั้นๆ จาก {{top_review_snippet}}

### 6. Price + Urgency (1-2 ประโยค)
"ราคา {{price}} บาท (ปกติ {{original_price}} ลด {{discount_percent}}%)"
เพิ่ม urgency แบบไม่เกินจริง: "ของลดราคาช่วงนี้" หรือ "เห็นของเหลือไม่เยอะ"

### 7. CTA + Link
```
สั่งซื้อ / ดูรายละเอียดเพิ่ม 👉 {{affiliate_link}}
```

### 8. Disclosure + Hashtags
```
.
.
#Shopeeaffiliate #[หมวดสินค้า] #[คำค้นยอดนิยม]
```
ใส่ . ก่อน hashtag เพื่อแยกบรรทัด ใช้ 3-5 hashtag พอ

## Tone
- เป็นกันเอง เหมือนเพื่อนแนะนำ ไม่ใช่ขายตรง
- ใช้ "เรา/ฉัน/ผม" ได้
- emoji สอดแทรกพอดี ไม่ฟุ่มเฟือย
- ห้าม CAPS LOCK ห้าม !!!!! เกิน 2 ตัว

## Hashtag Bank ตามหมวด
- **อิเล็กทรอนิกส์:** #ของมันต้องมี #ไอเทมเด็ด #รีวิวของดี
- **บิวตี้:** #รีวิวบิวตี้ #ของแม่ค้าแนะนำ #สกินแคร์
- **บ้าน:** #ของใช้ในบ้าน #ไอเทมแม่บ้าน #ของแต่งบ้าน
- **แฟชั่น:** #เสื้อผ้าราคาถูก #แต่งตัวงบน้อย
- **กิน/ดื่ม:** #ของกินอร่อย #ขนมเด็ด

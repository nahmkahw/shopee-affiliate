# Raw Sources

วางไฟล์ source ที่นี่ — articles, papers, notes, transcripts

**กฎ: LLM อ่านได้อย่างเดียว ห้ามแก้ไขไฟล์ในโฟลเดอร์นี้**

## วิธีเพิ่ม source

1. วางไฟล์ markdown (หรือ PDF, text) ในโฟลเดอร์นี้
2. รูปภาพให้วางใน `assets/`
3. บอก LLM: "ingest `raw/<filename>`"

## Tips

- ใช้ **Obsidian Web Clipper** (browser extension) แปลง web article เป็น markdown อัตโนมัติ
- หลัง clip article กด hotkey เพื่อ download images มาเก็บใน `assets/` (Obsidian Settings → Hotkeys → "Download attachments")
- ชื่อไฟล์แนะนำ: `YYYY-MM-DD-title-slug.md`
